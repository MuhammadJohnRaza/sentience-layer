# Sentience Layer - Mobile UI

Futuristic dark theme mobile interface with cyberpunk aesthetic.

## Features

- 🔥 Neon colors (red, orange, amber)
- ✨ Glassmorphism cards with backdrop blur
- 💫 Glowing hover effects
- 🌊 Floating particle animations
- 🎨 Dark mode with gradient backgrounds
- ⚡ Built with Next.js 14 & TypeScript
- 🎯 Tailwind CSS for styling

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3001](http://localhost:3001) to view the app.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide React Icons
- Framer Motion

## Design System

### Colors
- Neon Red: `#FF0040`
- Neon Orange: `#FF6B00`
- Neon Amber: `#FFB800`
- Cyber Dark: `#0a0a0f`
- Cyber Darker: `#050508`

### Components
- `GlassCard` - Glassmorphism card with glow effects
- `NeonButton` - Glowing button with gradient
- `FloatingParticles` - Animated background particles
