## 🚀 Quick Start Commands

### Start Backend
```bash
cd backend/python
python -m uvicorn api.main:app --reload --host 0.0.0.0 --port 8000
```

### Start Mobile App
```bash
cd mobile
npm install
npm start
# Then press 'a' for Android, 'i' for iOS, or 'w' for Web
```

### Test the System
1. Open app → Tap **"Content→Action"** (featured card with ⭐)
2. Select example: **"Business Report Analysis"**
3. Tap **"ANALYZE TEXT"**
4. View insights and reasoning chain
5. Tap **"SIMULATE"** on recommended action
6. Review simulation results
7. Tap **"EXECUTE ACTION"**
